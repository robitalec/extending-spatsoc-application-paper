3.  Path similarity indices
	- According to Duda_2001, "a metric distance function (d) is one that computes a generalized scalar distance between two objects while satisfying the following four properties"
			- Nonnegativity: d(x, y) >= 0
			- Reflexivity: d(x, y) = 0, IFF x = y
			- Symmetry: d(x, y) = d(y, x)
			- Triangle inequality: d(x, z) <= d(x, y) + d(y, z)
	- Metrics
		- Euclidean measurement
			- time-weighted distance metric (Sinha_2005)
			- Hausdorff distance is a shape comparison metric to evaluate the similarity between two point set (Huttenlocher_1993)
			- Modified version of the Huttenlocher_1993 given in Atev_2006 and Shao_2010 to consider order of points
			- Frechet distance (Alt_1995)
				- Discretized form (Eiter_1994)
			- Longest common subsequences LCSS (Vlachos_2002)
