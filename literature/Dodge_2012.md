Dodge, Somayeh, Patrick Laube, and Robert Weibel. 2012. “Movement Similarity Assessment Using Symbolic Representation of Trajectories.” _International Journal of Geographical Information Science_ 26 (9): 1563–88. [https://doi.org/10.1080/13658816.2011.630003](https://doi.org/10.1080/13658816.2011.630003).

---
Data
- hurricanes
- couriers

Proposes a modified version of edit distance: normalized weighted edit distance

Measures of  similarity between trajectories
- spatial similarity
	- similar geometric shape
	- ignores temporal dimension
- spatio-temporal similarity
	- both spatial and temporal aspects

time series similarity measures
- longest common subsequence LCSS
- dynamic time warping DTW
geometry shape matching
- Frechet distance
- Hausdorff distance
